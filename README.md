# Simple Context Menu Search Engines

## Description
Simple *chromium* based extension, adds ability to search highlighted text in specific search engine via browser context menu.

## Search Engines
	__Google__
	__Bing__
	__Yahoo!__
	__DuckDuckGo__
	__Yandex__
	__Brave__
	__Baidu__
